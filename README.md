## 📖 Course Description

This repository contains materials and **in-class exercises** developed during the lectures of the **Non-Relational Databases** course offered at PUCP for Master Students during the 2026-1 academic semester.

The course is of a theoretical-practical nature, whose purpose is for the student to understand the implementation and application of NoSQL databases. Topics developed include key-value stores, document databases, column-family stores, graph databases, distributed systems, and data modeling for non-relational environments.

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following tools installed:

- **Git** ≥ 2.30
- **Docker** (recommended for running NoSQL database instances like Redis, MongoDB, Cassandra, etc.)
- Appropriate IDE or text editor

### Clone the Repository

```bash
git clone https://github.com/jlcorcuera/pucp-non-relational-db-2026-01.git
cd pucp-non-relational-db-2026-01
```
